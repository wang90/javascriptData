## 手机号

#### 中国手机号(严谨), 根据工信部2019年最新公布的手机号段
````
/^(?:(?:\+|00)86)?1(?:(?:3[\d])|(?:4[5-7|9])|(?:5[0-3|5-9])|(?:6[5-7])|(?:7[0-8])|(?:8[\d])|(?:9[1|8|9]))\d{8}$/
````
#### 中国手机号(宽松), 只要是13,14,15,16,17,18,19开头即可
`````
/^(?:(?:\+|00)86)?1[3-9]\d{9}$/
`````
#### 中国手机号(最宽松), 只要是1开头即可, 如果你的手机号是用来接收短信, 优先建议选择这一条
`````
/^(?:(?:\+|00)86)?1\d{10}$/
`````

## 网址
`````
/^(?:(?:https?|ftp):\/\/)?(?:[\da-z.-]+)\.(?:[a-z.]{2,6})(?:\/\w\.-]*)*\/?/
`````
